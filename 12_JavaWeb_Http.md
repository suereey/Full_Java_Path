# Http
## What is Http: 超文本传输协议
- http: 是一个简单的请求－响应协议，它通常运行在TCP之上
    - 文本： html, 字符串
    - 超文本：图片，音乐，视频，定位，地图
    - port: 80
    - https port: 443
- 两个时代
    - http1.0: 
        - HTTP/1.0: 客户端可以与web服务器连接后，只能获得一个web资源. 断开连接

    - http2.0: 
        - HTTP/1.1: 客户端可以与web服务器连接后，可以获得多个web资源
